# Как получить готовый switch-wizard.exe без Windows-машины

Сборка идёт на бесплатном Windows-сервере GitHub (GitHub Actions). Вам нужен
только аккаунт GitHub — ни Windows, ни Python на вашем Mac не требуется.
Файл `.github/workflows/build-exe.yml` уже лежит в проекте и всё делает сам.

## Шаги (один раз, ~10 минут)

1. Зайдите на https://github.com и создайте новый репозиторий
   (кнопка **New**), например `switch-wizard`. Можно приватный.

2. Загрузите файлы проекта в репозиторий. Самый простой способ без командной
   строки: на странице пустого репозитория нажмите
   **uploading an existing file**, перетащите ВСЕ файлы проекта, включая папку
   `.github`, и нажмите **Commit changes**.

   Через терминал (если удобнее):

   ```bash
   cd папка-с-проектом
   git init
   git add .
   git commit -m "switch wizard"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГИН/switch-wizard.git
   git push -u origin main
   ```

3. После загрузки сборка запустится автоматически. Откройте вкладку
   **Actions** в репозитории — там будет запуск **build-exe** с крутящимся
   индикатором. Через 2–4 минуты он станет зелёным.

4. Кликните на завершённый запуск и внизу, в разделе **Artifacts**, скачайте
   **switch-wizard-windows** — это ZIP с `switch-wizard.exe` и `profiles.yaml`.
   Готово: распакуйте на любой Windows-ПК и запускайте.

## Как пересобрать после правок

Любой новый пуш в репозиторий (или кнопка **Run workflow** во вкладке
**Actions**) пересобирает `.exe` заново. Правки в `profiles.yaml`, кстати,
пересборки не требуют — файл читается рядом с `.exe` при запуске; пересборка
нужна только при изменении кода `.py`.

## Раздача операторам одной ссылкой (необязательно)

Если создать тег версии, GitHub автоматически приложит `.exe` к странице
**Releases** — оттуда удобно давать ссылку исполнителям:

```bash
git tag v1.0
git push origin v1.0
```

---

### Альтернатива: сборка на своей Windows-машине

Если под рукой есть Windows-ПК (свой или коллеги) — можно обойтись без GitHub:
положите файлы проекта в одну папку и запустите `build.bat` двойным кликом.
Результат — `dist\switch-wizard.exe` рядом с `dist\profiles.yaml`.
Подробности — в основном `README.md`.
